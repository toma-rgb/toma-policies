Privacy Policy

Last updated: 2025-12-23

This policy explains how the “voiceflow_bridge_app” (the “App”) processes data received via the WhatsApp Cloud API to enable automated replies through Voiceflow.

Data We Process

  • WhatsApp message content you send to the WhatsApp test/Business number.
  • WhatsApp metadata provided by Meta (e.g., sender number, timestamps, message IDs).

Purpose of Processing

We process messages solely to deliver automated responses and run conversation flows (Voiceflow). We do not sell data or use it for advertising.

Data Sharing

Message content is relayed to Voiceflow’s runtime API to generate responses. No other third parties receive data.

Retention

Sessions are cleared automatically after a short period or if you request deletion. Logs may retain minimal, aggregated technical information for troubleshooting.

Your Rights

You may request access or deletion of your data by emailing adamiatoma@gmail.com.
